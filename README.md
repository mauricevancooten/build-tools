# Sass

Sass examples use node-sass and Eyeglass to import Sass libraries.

## Dependencies

Node.js

## Instructions

1. Install Node modules `npm install`
2. Compile SCSS to CSS `npm run build`
3. Watch for changes and compile SCSS to CSS `npm run build:watch`
